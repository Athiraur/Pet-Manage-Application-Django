# Title
Pet-Manage-Application-Django

# Project Description

The Pet Management Application is a web-based system developed using Django. It helps manage pet details such as registration, owner information, and basic records in an organized way.

# Features

* Add new pets
* View pet details
* Update pet information
* Delete pet records
* Manage pet owner details

# Technologies Used

* Python
* Django
* MySQL
* HTML, CSS

# Installation Steps

# 1. Clone the repository

git clone https://github.com/your-username/pet-management.git
cd pet-management

# 2. Create virtual environment

python -m venv venv

# 3. Activate virtual environment

venv\Scripts\activate

# 4. Install dependencies

pip install -r requirements.txt

# 5. Configure database

Update `settings.py`:

# 6. Run migrations

python manage.py makemigrations
python manage.py migrate

# 7. Run server

python manage.py runserver

# 8. Usage

Open browser and go to:
http://127.0.0.1:8000/

# Project Structure

* `models.py` → Database models
* `views.py` → Business logic
* `templates/` → HTML pages
